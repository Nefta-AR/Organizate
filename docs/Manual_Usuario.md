# 1. IntroducciГіn

## 1.1 ВҝQuГ© es Simple?

**Simple** *(anteriormente OrganГӯzate)* es una aplicaciГіn mГіvil diseГұada como **prГіtesis cognitiva** para personas con condiciones del neurodesarrollo, principalmente:

- **TEA** (Trastorno del Espectro Autista)
- **TDAH** (Trastorno por DГ©ficit de AtenciГіn e Hiperactividad)

La app combina herramientas de organizaciГіn, comunicaciГіn aumentativa y alternativa (CAA), gestiГіn del tiempo y apoyo de inteligencia artificial en una sola plataforma configurable.

---

## 1.2 ВҝPara quiГ©n estГЎ diseГұada?

| Perfil | Funciones principales |
|--------|----------------------|
| рҹ‘Ө **Usuario TEA** | Tablero de pictogramas (AAC), rutinas visuales, TTS, contacto de emergencia |
| рҹ‘Ө **Usuario TDAH** | Lista de tareas, temporizador Pomodoro, desglose de tareas con IA (SГәper Experto), racha diaria |
| рҹ‘Ө **Usuario Mixto** | Acceso a todas las funciones, configurables desde Ajustes |
| рҹ‘ЁвҖҚрҹҸ« **Tutor** | SupervisiГіn remota de tareas, pictogramas, progreso y configuraciГіn del usuario vinculado |

---

## 1.3 Requisitos del sistema

- **Sistema operativo:** Android 8.0 (API 26) o superior
- **ConexiГіn a internet:** Requerida para autenticaciГіn, sincronizaciГіn y funciones IA
- **Permisos necesarios:**
  - рҹ“· CГЎmara (para crear pictogramas personalizados)
  - рҹ”” Notificaciones (para recordatorios de tareas y Pomodoro)
  - рҹ”Ӣ Ignorar optimizaciГіn de baterГӯa (para alarmas exactas)
- **Espacio libre:** MГӯnimo 150 MB
